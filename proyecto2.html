📊 Dashboard de Ventas

Dashboard interactivo desarrollado con Python, Pandas y Streamlit para analizar información de ventas mediante indicadores, filtros dinámicos y visualizaciones.

📌 Descripción

Se desarrolló una aplicación web interactiva utilizando Streamlit que permite cargar archivos CSV con información de ventas, calcular indicadores clave (KPIs), aplicar filtros y visualizar los datos mediante gráficos dinámicos. El proyecto fue diseñado para facilitar el análisis comercial y apoyar la toma de decisiones basada en datos.

🎯 Objetivo

Automatizar el análisis de documentos financieros para facilitar la consulta de información relevante de forma rápida y precisa.

❗ Problema

Muchas pequeñas empresas almacenan su información de ventas en archivos CSV o Excel, pero carecen de herramientas que permitan analizar esos datos de forma rápida e intuitiva. Esto dificulta identificar tendencias, comparar resultados entre ciudades o categorías y obtener indicadores relevantes para la toma de decisiones.

💡 Solución

Se implementó un dashboard interactivo en Streamlit capaz de importar archivos CSV, calcular automáticamente métricas de ventas y generar visualizaciones dinámicas. La aplicación permite explorar la información mediante filtros y gráficos, proporcionando una herramienta sencilla para el análisis comercial.

📈 Resultados obtenidos

  • ✅ Automatización del análisis de estados financieros en formato PDF.
  • ✅ Respuesta automática a preguntas en lenguaje natural.
  • ✅ Extracción precisa de información financiera mediante IA.
  • ✅ Reducción del tiempo de consulta de información relevante.
  • ✅ Integración exitosa entre n8n y Google Gemini.
