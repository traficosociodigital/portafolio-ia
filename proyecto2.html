📊 Dashboard de Ventas

Python • Pandas • Streamlit

Dashboard interactivo para el análisis de ventas mediante indicadores, filtros dinámicos y visualización de datos.

📌 Descripción

Se desarrolló una aplicación web utilizando Streamlit que permite cargar archivos CSV con información de ventas, calcular indicadores clave (KPIs), aplicar filtros y visualizar los datos mediante gráficos interactivos para facilitar el análisis comercial.

🎯 Objetivo

Desarrollar un dashboard interactivo que permita analizar información de ventas mediante indicadores, filtros y visualizaciones dinámicas para apoyar la toma de decisiones basada en datos.

❗ Problema

Muchas empresas almacenan su información comercial en archivos CSV o Excel, pero no cuentan con herramientas que permitan visualizar rápidamente el comportamiento de las ventas y obtener indicadores relevantes para la toma de decisiones.

💡 Solución

Se implementó un dashboard interactivo en Streamlit que permite importar archivos CSV, calcular automáticamente indicadores de ventas y generar gráficos dinámicos con filtros por ciudad y categoría.

📈 Resultados obtenidos

  • ✅ Desarrollo de un dashboard web interactivo.
  • ✅ Cálculo automático de indicadores (KPIs).
  • ✅ Visualización dinámica de ventas mediante gráficos.
  • ✅ Implementación de filtros por ciudad y categoría.
  • ✅ Carga dinámica de archivos CSV.
  • ✅ Interfaz desarrollada completamente en Streamlit.
